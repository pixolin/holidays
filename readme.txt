Plugin ads custom post type "holidays" and a user "tourist" with limited capabilities.

== Description ==

Adds user "tourist" with limited capabilities upon plugin activation.

Adds custom post type "holidays". Capabilities are granted to user "tourist", "admin" and "editor".

== Installation ==

1. Upload folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
